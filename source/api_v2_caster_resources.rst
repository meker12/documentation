Caster Resources
================

Get details on all casters belonging to this account
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   GET /casters/

Request/Response:

.. code-block:: bash

   $ ERROR

Get details on a caster specified by ID
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   GET /casters/<caster_id>/

Request/Response:

.. code-block:: bash

   $ ERROR
