Remote Resources
================

Get details on an account's remote instances.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   GET /remote_instances/

Request/Response:

.. code-block:: bash

   $ http

Get details on an account's remote instances.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   GET /remote_instances/<instance_name>/

Request/Response:

.. code-block:: bash

   $ http

Rename the specified remote instance.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   POST /remote_instances/<instance_name>/rename/

Request/Response:

.. code-block:: bash

   $ http
