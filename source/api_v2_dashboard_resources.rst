Dashboard Resources
===================

Get all dashboards for the specified account
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   GET /dashboards/

Request/Response:

.. code-block:: bash

   $ ERROR

Create a new dashboard
~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   POST /dashboards/

Request/Response:

.. code-block:: bash

   $ ERROR

Get the dashboard specified by ID
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   GET /dashboards/<dashboard_id>/

Request/Response:

.. code-block:: bash

   $ ERROR

Update the dashboard specified by ID
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   PUT /dashboards/<dashboard_id>/

Request/Response:

.. code-block:: bash

   $ ERROR

Delete the dashboard specified by ID
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   DELETE /dashboards/<dashboard_id>/

Request/Response:

.. code-block:: bash

   $ ERROR
